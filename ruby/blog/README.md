# Blog app

## First setup
In terminal:
```
git clone https://github.com/ReDI-School/group1-blog.git
cd group1-blog
bundle install
rake db:migrate
rake db:seed
rails server
```

## Stages
* v0.1 - Post model, Post controller, show and index views
* v0.2 - Add new view and logic to create a new post
